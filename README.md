# Tamil Text Summarization using NLP

This project implements extractive text summarization for Tamil language using basic Natural Language Processing techniques.

## Steps
1. Text Cleaning
2. Sentence Tokenization
3. Word Tokenization
4. Stopword Removal
5. Word Frequency Calculation
6. Sentence Scoring
7. Summary Generation

## Tools Used
- Python
- Jupyter Notebook
- Google Colab

## Input
Tamil biography text of Dr. A. P. J. Abdul Kalam

## Output
Concise summary generated from the original text.
